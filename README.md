# Randel et al. — Reproducible Analysis Code

This repository contains the code and data access instructions required to reproduce the analyses and results reported in **Randel et al.** The repository is organized by experimental modality and provides the processing and analysis pipelines used in the study.

## Overview

The codebase supports the reproduction of the main findings presented in the manuscript. Where datasets are too large or subject to distribution restrictions, this repository provides links and instructions for obtaining the necessary data.

## Repository Structure

```
.
├── light-sheet/
├── behaviour/
└── README.md
```

### `light-sheet/`

Code for processing, analysis, and visualization of light-sheet imaging data.

### `behaviour/`

Code for behavioral data preprocessing, analysis, and figure generation.

## Data Availability

Data required to reproduce the analyses can be obtained through the links and instructions provided within the relevant subdirectories. Please refer to the documentation in each module for dataset-specific requirements and download procedures.

## Reproducing the Results

Detailed instructions for reproducing individual analyses are provided in the corresponding subdirectory documentation. In general, reproduction involves:

1. Obtaining the required datasets.
2. Installing the software dependencies.
3. Running the preprocessing pipelines.
4. Executing the analysis scripts.
5. Generating figures and summary statistics.
